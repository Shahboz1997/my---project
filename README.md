Сайт онлайн-займов (незавершенный проект)
Описание:
Разработан аналог интерфейса российского сайта по выдаче микрозаймов. Проект включает в себя основные элементы:
Меню навигации;
Форму заявки на получение займа;
Форму получения денежных средств;
Список предложений (условий займа);
Бургер-меню, реализованное на JavaScript;
Адаптивный дизайн (частично).
Проект требует доработки и завершения.
Инструкции для локального запуска
Для запуска проекта на вашем локальном компьютере выполните следующие шаги в терминале или командной строке, находясь в корневой директории проекта:
Для локалного запуска
1. npm i
2. npm run start
